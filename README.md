## 사용방법
본 README는 윈도우 환경을 기준으로 작성하였습니다.

- [여기서](https://nodejs.org) nodejs 17 ~ 18버전을 다운로드 해주세요.
- 그리고 install.bat을 실행해주세요.
- .env의 TOKEN 항목을 봇 토큰으로 바꿔주세요.
- 터미널에서 node index.js를 실행하시면 봇이 실행되요!
